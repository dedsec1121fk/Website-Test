Termux Basics: Turn Your Phone Into a Linux Machine

Learn how to transform your Android device into a powerful Linux terminal using Termux. Complete guide covering installation, basic commands, package management, and practical uses for security professionals and enthusiasts.

Termux & Hacking 12 min read 2026-01-13

What Is Termux and Why Should You Care?

Termux is a powerful terminal emulator and Linux environment application for Android. Unlike other terminal apps that merely provide a shell interface, Termux creates a complete Linux environment on your phone without requiring root access. This means you can run real Linux commands, install packages, write scripts, and even run servers—all from your pocket.

For security professionals, ethical hackers, and technology enthusiasts, Termux opens up a world of possibilities. You can practice Linux commands on the go, test network tools, develop scripts, and learn cybersecurity concepts anywhere. Whether you're waiting for a bus or sitting in a coffee shop, your Android device becomes a legitimate hacking workstation.

Installing Termux: The Right Way

The first critical step is obtaining Termux from the correct source. While Termux was previously available on the Google Play Store, the version there is now outdated and no longer maintained. The official and recommended source is F-Droid, an open-source app repository that hosts the latest Termux builds.

Step-by-Step Installation

First, download F-Droid from f-droid.org on your Android device. Once installed, open F-Droid and search for 'Termux'. Install the main Termux app along with Termux:API for accessing device features and Termux:Styling for customization. After installation, open Termux and let it complete its initial setup, which downloads the base system packages.

The very first command you should run after opening Termux is updating the package repositories and upgrading existing packages. This ensures you have the latest security patches and software versions available:

pkg update && pkg upgrade

Essential Linux Commands for Beginners

The Linux command line might seem intimidating at first, but understanding a handful of essential commands will give you the foundation to navigate and control your Termux environment effectively. Let's explore the most important commands every beginner should master.

Navigating the File System

The pwd command (print working directory) shows your current location in the file system. When you first open Termux, you'll be in your home directory, typically displayed as /data/data/com.termux/files/home or simply ~ (tilde).

pwd
# Output: /data/data/com.termux/files/home

The ls command lists files and directories. Adding flags like -la shows detailed information including permissions, ownership, and hidden files (those starting with a dot):

ls -la
# Shows all files with detailed permissions

Use cd (change directory) to move between directories. You can use absolute paths starting from root, or relative paths from your current location:

cd /sdcard       # Move to external storage
cd ..            # Move up one directory
cd ~             # Return to home directory
cd -             # Go to previous directory

File Operations

Creating, copying, moving, and deleting files are fundamental operations. The touch command creates empty files, while mkdir creates directories. Use cp to copy and mv to move or rename files:

touch myfile.txt          # Create empty file
mkdir projects            # Create directory
mkdir -p path/to/nested   # Create nested directories
cp file.txt backup.txt    # Copy file
mv old.txt new.txt        # Rename or move file
rm file.txt               # Delete file (careful!)
rm -rf directory/         # Delete directory and contents

Warning: The rm command permanently deletes files without sending them to a trash folder. The -rf flags force deletion of directories and all their contents without asking for confirmation. Use with extreme caution.

Viewing and Editing Files

Several commands help you view file contents. The cat command displays entire file contents, while less and more allow paginated viewing of longer files. For quick previews, head and tail show the first or last lines respectively:

cat file.txt              # Display file contents
less file.txt             # Paginated view (q to quit)
head -20 file.txt         # First 20 lines
tail -f logfile.txt       # Follow file updates in real-time

For editing files directly in the terminal, install and use nano, a beginner-friendly text editor:

pkg install nano
nano myfile.txt           # Edit file (Ctrl+X to exit)

Package Management with pkg

Termux uses the pkg command (a wrapper around apt) for package management. This powerful system allows you to install thousands of packages including programming languages, security tools, development utilities, and more.

Essential Package Commands

pkg update                    # Update package lists
pkg upgrade                   # Upgrade installed packages
pkg install package-name      # Install a package
pkg uninstall package-name    # Remove a package
pkg search keyword            # Search for packages
pkg list-installed            # Show installed packages

Must-Have Packages for Beginners

Here are essential packages every Termux user should consider installing. These provide core functionality for development, networking, and system administration:

pkg install git               # Version control
pkg install python            # Python programming
pkg install nodejs            # JavaScript runtime
pkg install openssh           # SSH client and server
pkg install curl wget         # Download utilities
pkg install nmap              # Network scanner
pkg install vim nano          # Text editors
pkg install htop              # Process viewer
pkg install net-tools         # Network utilities

Accessing Phone Storage

By default, Termux operates in its own isolated environment. To access your phone's internal storage, photos, downloads, and other files, you need to grant storage permissions:

termux-setup-storage

After running this command and granting permission, you'll find a new storage directory in your home folder with symbolic links to various phone directories:

ls ~/storage/
# dcim/  downloads/  movies/  music/  pictures/  shared/

Setting Up SSH Access

One of the most powerful features of Termux is the ability to run an SSH server, allowing you to connect to your phone from a computer. This is invaluable for typing on a real keyboard while using your phone's Linux environment.

pkg install openssh
passwd                        # Set a password
sshd                          # Start SSH server
whoami                        # Get your username
ifconfig                      # Get your IP address

From your computer, connect using: ssh username@phone-ip -p 8022 (Termux uses port 8022 by default instead of the standard port 22).

Customizing Your Environment

A well-configured environment makes working in Termux more efficient and enjoyable. Start by installing helpful utilities and customizing your shell:

Installing Oh My Zsh

pkg install zsh curl git
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

Creating Aliases

Aliases are shortcuts for frequently used commands. Add them to your .bashrc or .zshrc file:

echo "alias update='pkg update && pkg upgrade'" >> ~/.bashrc
echo "alias ll='ls -la'" >> ~/.bashrc
echo "alias c='clear'" >> ~/.bashrc
source ~/.bashrc

Practical Projects to Get Started

The best way to learn is by doing. Here are some beginner-friendly projects that will help you practice and understand the power of having Linux in your pocket:

Project 1: Create a Simple Bash Script

nano backup.sh
# Add the following content:
#!/bin/bash
echo "Creating backup..."
cp -r ~/documents ~/backups/$(date +%Y%m%d)
echo "Backup complete!"

chmod +x backup.sh
./backup.sh

Project 2: Set Up a Local Web Server

pkg install python
mkdir mywebsite && cd mywebsite
echo "<h1>Hello from Termux!</h1>" > index.html
python -m http.server 8080
# Access at http://localhost:8080

Project 3: Clone and Explore a GitHub Repository

pkg install git
git clone https://github.com/termux/termux-app
cd termux-app
ls -la
cat README.md | less

Security Considerations

While Termux provides powerful capabilities, it's important to use it responsibly and securely. Never download and run scripts from untrusted sources without examining them first. When using network tools, only test on networks you own or have explicit permission to test. Keep your packages updated to receive security patches, and be cautious about granting Termux permissions to sensitive data.

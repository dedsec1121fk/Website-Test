1) Install Termux the right way

# Prefer F-Droid (or GitHub releases) for the newest Termux
# After install:
pkg update -y && pkg upgrade -y

Avoid outdated Play Store builds. After the first update/upgrade, Termux becomes stable and you can install tools normally.

2) Give storage access (so you can save files)

termux-setup-storage

This creates /storage/emulated/0/ access and a ~/storage shortcut. Use it to read/write Downloads, Documents, etc.

  • Your Termux home is: /data/data/com.termux/files/home (use ~)
  • Downloads path (easy): ~/storage/downloads

3) Packages vs pip (don’t mix blindly)

Use pkg for system tools (git, openssh, clang, curl). Use pip for Python libraries. If something fails, install the system dependency with pkg first.

pkg install -y git openssh curl
pkg install -y python
python -m pip install --upgrade pip

4) A clean folder layout for projects

Create one Projects folder and keep scripts inside it. This prevents chaos and makes backups simple.

mkdir -p ~/Projects
cd ~/Projects
  • Use names like: notes/, scraper/, bot/
  • Keep a README.txt in each project

5) Daily workflow that actually works

A good routine is: update weekly, code daily, and back up monthly. Small consistency beats rare big sessions.

  • Weekly: pkg update && pkg upgrade
  • Daily: 20–40 minutes on one small goal
  • Monthly: copy ~/Projects to Downloads or cloud

6) Safety basics (so you don’t get burned)

Before running any script, read it. If it asks for strange permissions, deletes files, or downloads random binaries, stop and review. Use GitHub repos you trust and pin versions.

7) Next step: what to learn first

If you learn Linux basics (cd/ls/cp/mv), git basics (clone/pull), and Python basics (variables, loops, functions), you can build almost any Termux tool.

pkg update -y && pkg upgrade -y
termux-setup-storage
pkg install -y git python
mkdir -p ~/Projects && cd ~/Projects
python --version

cd ~/Projects/DedSec
git pull
python tool.py
cp output.txt /storage/emulated/0/Download/
nano NOTES.md

pkg search ffmpeg
pkg install ffmpeg

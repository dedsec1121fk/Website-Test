FAQ

Quick answers to the most common questions.

Is this legal?

The project focuses on educational and ethical use: defense, privacy, automation, and learning. You must follow laws and respect consent.

Do I need root?

Most Termux workflows do NOT require root. Root is optional for certain advanced cases, but this site prioritizes non-root solutions.

How do I install Termux safely?

Use the official Termux releases (typically via F-Droid). Then update packages and enable storage. See the installation guide page.

Installation guide
Where are scripts saved?

It depends on the tool. A common default is Termux home (~) or the Android Downloads folder after running: termux-setup-storage.

Can I collaborate?

Yes. We can collaborate on tools, documentation, design, translations, or testing.

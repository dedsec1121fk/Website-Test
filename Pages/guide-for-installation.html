Guide for Installation

A clean, beginner-friendly setup for Termux on Android.

1) Install Termux (recommended)

Prefer official builds (commonly via F-Droid). Avoid random APK sites.

2) Update packages

Run these once after installing Termux. 

pkg update -y
pkg upgrade -y

3) Enable storage access

This allows scripts to read/write in internal storage (Downloads, etc.). 

termux-setup-storage

4) Install common tools

A small baseline for most workflows. 

pkg install -y python git openssh curl wget nano

Recommended Python setup

Install pip packages safely and keep projects separated.

Create a virtual environment

python -m venv venv
source venv/bin/activate

Install packages

pip install --upgrade pip
pip install requests rich

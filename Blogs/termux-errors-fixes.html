Termux Errors and How to Solve Them

Common errors explained with quick fixes and commands.

← Back to Blog Install DedSec Project

First steps when something fails

Most issues are fixed by updating packages and cleaning broken installs: 

pkg update -y && pkg upgrade -y
pkg --check-mirror update

If you installed a lot of things and it’s messy, you can reinstall a package: 

pkg reinstall python

ModuleNotFoundError (Python)

It means a Python library is missing. Install it with pip: 

python -m pip install --upgrade pip
python -m pip install requests

If a package needs compilation, install build tools: 

pkg install -y clang make pkg-config

Permission denied / can't access storage

Give Termux storage permission and mount shared storage: 

termux-setup-storage

Then use: ~/storage/shared/Download

'E: Unable to locate package' or repo errors

Switch mirrors and update: 

termux-change-repo
pkg update -y

If you’re using an old Termux build, install the F-Droid version.

No space left on device

Check disk usage and remove caches: 

df -h
pkg clean

Also delete old downloads in ~/storage/shared/Download.

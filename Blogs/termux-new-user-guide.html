Install Termux the right way

Use the F-Droid build (recommended) so updates work smoothly. After installing, run the basic update: 

pkg update -y && pkg upgrade -y

Then give storage access (so Termux can read/write to Downloads): 

termux-setup-storage

Your shared storage will appear at: 

~/storage/shared

Essential packages for daily use

These are safe, useful, and cover 90% of beginner needs: 

pkg install -y git curl wget nano vim unzip zip openssh python
  • git: download and manage projects
  • curl/wget: download files
  • openssh: ssh/sftp to servers & PCs
  • python: scripting & automation

Folder basics (so you don't get lost)

Termux starts in your home folder (~). Learn these 6 commands: 

pwd
ls
cd Downloads
mkdir Projects
cd ~/Projects
rm -r folder_name

Pro tip: keep your coding projects inside ~/Projects (faster, cleaner).

Good habits (performance + safety)

  • Back up important folders (Projects, scripts).
